# 빅데이터의이해

## 개요
빅데이터 시대에는 데이터의 양이 급증하고 새로운 형태의 데이터가 일상화되면서 데이터로부터 다양한 가치와 통찰력을 얻고 있다. 이 강의는 빅데이터를 개념, 기술, 활용, 전달, 문제점 등을 포괄적으로 살펴보는 것을 목적으로 한다.
